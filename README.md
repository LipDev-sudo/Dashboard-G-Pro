# Dashboard G-Pro

Dashboard de gestao profissional com interface moderna e visualizacao de dados em tempo real.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Painel de metricas e KPIs
- Graficos interativos
- Layout responsivo
- Interface limpa e intuitiva
- Componentes reutilizaveis
